Arquitectura cliente-servidor

¿Que es la arquitectura Cliente-Servidor?

El cliente y el servidor son 2 computadoras. El cliente es quien pide informacion y el servidor es quien entrega la informacion que pide el cliente, siempre y cuando el cliente tenga los permisos.

Cada solicitud que hace el cliente se llama peticion(request).

Front-End

¿Que es el Front-End?

Son los que desarollan del lado del cliente

Un programador front-end debe diseñar ademas de programar.

¿Cuales son los roles dentro del Front-End?

Diseñador IU

Maquetador

Programador Front-End

¿Cual es la función de cada uno de los roles del Front End

Diseñador IU: Diseña los flujos de usuario, las pantallas e interacciones en un programa de diseño (no escribe codigo).

Maquetador: Lleva el diseño a codigo HTML y CSS. No se preocupa de la logica, solo de la presentacion.

Programador Front-End: Agrega datos reales desde una API y logica a la presentacion creada por el maquetador.

BackEnd

¿Que es el backEnd?

Son los que desarollan del lado del servidor

¿Cuales son los roles del BackEnd?

Backend Developer

Database Administrator

Administradorr del Servidor

¿Cual es la función de cada rol?

Backend Developer: Se encarga de la logica del negocio y crea las API's para que el front-end pueda consumirlas.

Database Administrator: Diseña, implementa, mejora y mantiene el sistema de base de datos.

Administradorr del Servidor: Gestiona la instalacion, soporte y mantiene el servidor en donde se aloja la web o app.

    • ¿Que es un Full-Stack?

    Los desarrolladores con este perfil tienen tanto los conocimientos para trabajar del lado del servidor (back-end); como las habilidades para entender la experiencia del usuario y anticiparse a sus necesidades (front-end).

    Ademas, cuenta con el conocimiento de un sistema de base de datos, un sistema de control de versiones y una metodología de desarrollo ágil; herramientas fundamentales para trabajar en un entorno de desarrollo.

    Ejemplos de Full-Stack

  • Lenguajes de desarrollo web (PHP, Ruby, Python, Java y .Net)
  • Herramientas como MySQL, Oracle y SQL Server
  • Software de control de versiones como SVN, CVS o Git
  • MEAN Stack (MongoDB, Express.js, Angular.js y Node.js)
  • MERN Stack (MongoDB, Express.js, React y Node.js)
  • MEVN Stack (MongoDB, Express.js, Vue.j y Node.js)
